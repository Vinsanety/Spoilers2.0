89th Academy Awards Oscar-Pick-Em
We Act like we know movies. We don't. But for the last 89 years we have played a game where each host has picked their own winner from each Academy Award category.

The game goes: the player with the most points gets to pick a penalty movie for the player with the least amount of points. The loser must host this penalty movie on a future epsiode of Spoilers! podcast.
